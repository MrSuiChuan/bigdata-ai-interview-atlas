# 说明

这页汇总 P0 项目的整理进度。P0 项目不是直接搬运到系统，而是整理成项目页、面试题和主知识库映射。

# 汇总

1. P0 项目数：24
2. 项目整理页：24
3. 项目面试题：120

# AI Agent

1. [hello-agents](./ai-agent/hello-agents)：从零开始构建智能体，适合沉淀 Agent 基础、工具调用、任务循环和项目实践表达。
1. [agentic-ai](./ai-agent/agentic-ai)：围绕 Agentic AI 课程内容整理规划、行动、反馈、工具使用和多步任务边界。
1. [agent-tutorial](./ai-agent/agent-tutorial)：适合做 Agent 入门路线和基础面试题素材，把概念、流程和最小实现串起来。
1. [self-harness](./ai-agent/self-harness)：聚焦长时间运行 Agent 的 Harness Engineering，适合整理可靠性、状态恢复、评估和观测。
1. [handy-multi-agent](./ai-agent/handy-multi-agent)：基于 CAMEL 多 Agent 教程，适合整理角色划分、通信、协同边界和成本控制。
1. [hugging-multi-agent](./ai-agent/hugging-multi-agent)：基于 MetaGPT 等多 Agent 思路，适合整理角色型协作、工作流产物和工程治理。
1. [easy-langent](./ai-agent/easy-langent)：围绕 LangChain、LangGraph、Lagent 等生态，适合整理框架选型和 runtime 能力对比。
1. [agent-skills-with-anthropic](./ai-agent/agent-skills-with-anthropic)：围绕 Agent Skills 课程整理 skill、tool、上下文工程和任务封装边界。
1. [hello-generic-agent](./ai-agent/hello-generic-agent)：适合整理通用 Agent、自进化能力边界和可配置 Agent 的工程风险。

# RAG

1. [all-in-rag](./rag/all-in-rag)：RAG 技术全栈指南，适合整理文档治理、检索、重排、生成、评估和生产优化。
1. [wow-rag](./rag/wow-rag)：RAG 框架和教程项目，适合整理可复用 RAG 管线、模块边界和框架化实践。
1. [llm-universe](./rag/llm-universe)：面向大模型应用开发，适合整理 RAG 应用、知识库问答、应用工程和端到端项目表达。
1. [what-is-vs](./rag/what-is-vs)：向量检索与 RAG 实践，适合整理向量表示、相似度、ANN、召回和检索边界。
1. [easy-vecdb](./rag/easy-vecdb)：向量数据库基础教程，适合整理向量库职责、索引、过滤、更新和 RAG 边界。

# 大模型

1. [happy-llm](./llm/happy-llm)：从零开始构建大模型，适合整理 LLM 学习路径、基础概念和训练到应用的主线。
1. [base-llm](./llm/base-llm)：从 NLP 到 LLM 的算法全栈教程，适合整理传统 NLP、Transformer 和 LLM 的演进关系。
1. [self-llm](./llm/self-llm)：开源大模型部署和微调实践，适合整理本地部署、LoRA、推理服务和资源约束。
1. [diy-llm](./llm/diy-llm)：系统性大语言模型构建课程，适合整理预训练数据、Tokenizer、Transformer、MoE、CUDA/Triton 和分布式训练。
1. [code-your-own-llm](./llm/code-your-own-llm)：端到端定义模型从零训练到工程落地，适合整理白盒实现和面试中的机制解释。
1. [tiny-universe](./llm/tiny-universe)：小模型和训练系统实践，适合整理小规模实验、训练闭环和系统边界。
1. [post-training-of-llms](./llm/post-training-of-llms)：围绕 LLM 后训练课程，适合整理 SFT、偏好优化、DPO/RLHF、评估和安全边界。
1. [llm-cookbook](./llm/llm-cookbook)：面向开发者的 LLM 入门教程，适合整理 API 调用、Prompt、RAG、工具和评估的应用开发路径。
1. [llm-deploy](./llm/llm-deploy)：大模型推理和部署理论与实践，适合整理推理引擎、显存、吞吐、延迟和生产部署。
1. [llms-from-scratch-cn](./llm/llms-from-scratch-cn)：从零构建大语言模型，适合整理 GLM/Llama/RWKV 类模型实现和从机制到代码的白盒理解。
